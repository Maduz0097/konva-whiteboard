# konva-whiteboard

A whiteboard app made with Konva library.

Demo: [Click Me](https://whiteboard.leohong.dev)

## Tech

- TypeScript
- React
- Next.js
- Redux
- Tailwind CSS
- MUI (Material UI)
- Konva.js

## Run

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
